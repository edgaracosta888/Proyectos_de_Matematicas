# Proyectos de Matemáticas

**Repositorio de exploración matemática computacional. Cada carpeta es un proyecto independiente dedicado a un problema o área específica — con sus propias herramientas, datos y documentación.**

---

## Proyectos

| Carpeta | Tema | Descripción |
|---------|------|-------------|
| [`Numeros_Primos/`](Numeros_Primos/) | Números primos | Toolkit de identificación y generación de primos, dataset de 3M+ primos, y exploración experimental en busca de patrones. |
| [`Collatz/`](Collatz/) | Conjetura de Collatz | Verificación de secuencias bajo la regla $3n+1$ y espacio para futuras exploraciones sobre la estructura de las trayectorias. |

Cada proyecto tiene su propio `README.md` con documentación detallada.

---
